# ArtBox

Projet PHP / MySQL OpenClassrooms.

## Installation

1. Copier le dossier dans :
C:\xampp\htdocs

2. Lancer XAMPP

3. Démarrer Apache et MySQL

4. Importer la base de données dans phpMyAdmin :
database/artbox.sql

5. Ouvrir le site :
http://localhost/artbox

## Technologies utilisées

- PHP
- MySQL
- HTML
- CSS
- PDO
- Git / GitHub
